# Portfolio & CV Interactif — Sandra Gregoire

Site web personnel (portfolio / CV en ligne) hébergé sur **GitHub Pages**.
Ce site présente mon parcours, mes compétences techniques et mes projets réalisés dans les domaines du **systemes/logiciel**, de la **robotique** et de la **R&D**.

---

## En ligne

**[Voir le site](https://sandrag03.github.io/Website/)**

---

## Fonctionnalites principales

- **Design responsive** — tel quel pour PC, tables et mobiles
- **Mode Terminal / Mode Relaxed** — design double pour differents usages
- **Portfolio de projets** — modals intrusis avec photos et details
- **Timeline d'experiences** — representation graphique du CV double formation et alternance
- **Formulaire de contact** integration pour envoi d'emails

---

## Technologies utilisées

- **HTML5** semantique
- **CSS3** (variables CSS, Flexbox, Grid, animations, responsive)
- **JavaScript** vanilla (interaction page, modals, lightbox, formulaire)
- **Aucun framework** — 100% natif

---

## Structure

```
index.html     -> Page d'accueil (Hero, biographie rapide)
about.html     -> Profil & CV
education.html -> Parcours academique & Formation
projects.html  -> Portfolio de projets detaille
contact.html   -> Contact avec formulaire
ambitions.html -> Expeditions internationales et projets perso
```

